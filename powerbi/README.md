## Power BI Dashboard

This folder contains Power BI assets for visualizing SOC security metrics.

### Dashboard Features
- Failed login attempts by user
- Security risk summary and insights
- Gold-layer data consumed directly from ADLS Gen2

### Purpose
- Enable security monitoring and analysis
- Provide actionable insights for SOC teams
- Support investigation of suspicious authentication patterns

### Data Source
- Azure Data Lake Storage Gen2 (Gold layer)
- Delta / Parquet-based analytical tables
