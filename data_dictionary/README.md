## Data Dictionary

This folder contains the data dictionary for the SOC Lakehouse project.

It defines the structure, data types, and business meaning of fields used across the Bronze, Silver, and Gold layers in the Databricks medallion architecture.

### Contents
- Source fields ingested from raw security logs
- Transformed and enriched fields in the Silver layer
- Aggregated and analytical fields in the Gold layer

### Purpose
- Ensure data clarity and consistency
- Support analytics, reporting, and security investigations
- Help stakeholders understand how security metrics are derived
